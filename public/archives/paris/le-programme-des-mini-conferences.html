Le programme des mini conférences

Publié le 01/03/2017

Nous vous communiquons le programme pour notre soirée des mini-conférences PHP. qui se déroulera le jeudi 26 janvier (dans 2 jours) :

 

  • Le PHP chez Deezer (Jean Pasdeloup)
  • Mkframework : un framework PHP différent avec un générateur web (Mika duPot)
  • K-mean clustering with PHP (Delespierre Benjamin)
  • Les supers Globale $_GET et $_POST (Gilles Kponkou)
  • Un code métier 'framework agnostic' (Mathieu LEMOINE)
  • Héritage de namespaces, et généralisation (Julien DELSESCAUX)
  • Développer pour une CivicTech (Aurélien DAVID)
  • Mail templating for Swift Mailer (Xavier HUBERTY)
  • Hoa, un ensemble de bibliothèques PHP qui roxx du poney (Alexis von Glasow)
  • Les logs (Frédéric Sagez)

 

Alors n’attendez pas, pour vous inscrire car le nombre de place est limité.

Ce rendez-vous a été rendu possible grâce Deezer pour le leu et  JL Recrutement pour le verre de l’amitié
