Headless CMS et élection du bureau de l’antenne locale

Publié le 21/06/2019

Jeudi 20 Juin 2019
Cantine Numérique de Tours,

Une présentation par Florian Montalbano et Sébastien Courjean pour faire connaître les Headless CMS.

Vous gérez les contenus de vos applications ou sites web sur différentes interfaces d’administration ? Une nouvelle solution est apparue depuis 3-4 ans : les Headless CMS.

Un back-office pour les gouverner tous
Un back-office pour les trouver
Un back-office pour les amener tous,
Et dans les API les lier

Découverte de la solution Cockpit avec une présentation de l’interface d’administration et des tests d’intégration front-end sur deux solutions basées sur Symfony 4.3.

La présentation s’est terminée avec beaucoup de questions et d’échanges, démontrant l’intérêt pour cette nouvelle solution.

Ensuite, nous avons eu l’honneur de recevoir le président de l’AFUP Adrien Gallou qui nous a parlé de :

  • l’AFUP
  • les antennes locales
  • les événements annuels : Forum PHP, AFUP Day
  • les élections

L’élection du bureau de l’antenne locale s’est alors déroulée avec la validation de la liste candidate : Mickaël Auger (reconduction), Florian Montalbano (reconduction), Sébastien COURJEAN (reconduction) et Antoine Obligis (nouvel arrivant)

Les sponsors de la soirée :

  • PALO ALTOURS : la salle pour partager un agréable moment
  • Apside TOP : des pizzas qui ont régalés les personnes présentes
  • Jetbrains : un heureux gagnant avec une licence d’un an offerte
  • Edition Eyrolles : un quiz et la victoire pour l’un des participant avec de la lecture pour l’écologie dans toutes les étapes de développement
