Prochain Meetup PHP #17 : Tester son application PHP

Publié le 13/11/2017

Ce prochain Meetup s’organisera autour de 3 talks de 20 minutes traitant d’un sujet commun : le « testing » de son application PHP. **
**

PREMIER TALK
20 minutes – JUNIOR

Julien Vitte (@pichart) : Tu peux pas test !

Les tests, tout le monde en parle, trop peu de projets les utilisent et surtout dans la durée. Essayons de casser les préjugés et de trouver des cas d’utilisation permettant la mise en place de stratégies de test efficaces.

SECOND TALK
** **20 minutes – MEDIOR

Mikael Letang (@mikael_letang) : Faire émerger son architecture via du TDD (Test-Driven Development) double boucle

La conception dirigée par les tests : une exploration.

DERNIER TALK
** **20 minutes – JUNIOR

Nicolas Perussel (@mamoot64) : Une approche du BDD
(Behavior-Driven Development) avec Behat

Behat est un framework vous permettant de jouer des tests fonctionnels à partir de scénarios que vous décrivez (sous forme textuelle). Nous verrons, comment en quelques minutes, intégrer Behat dans vos projets. Nous jouerons également quelques tests sous forme d’exploration afin de vous donnez les clés pour un apprentissage plus poussé de ce framework.

ECHANGE & CONVIVIALITE

Le Meetup se terminera, comme toujours, par un apéro collaboratif – Pensez à amener un petit quelque chose à boire et/ou grignoter !
