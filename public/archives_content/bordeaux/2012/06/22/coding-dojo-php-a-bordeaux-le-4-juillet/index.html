Coding Dojo PHP à Bordeaux le 4 Juillet

Publié le 22/06/2012

Pour fêter l’arrivée de l’été, le PUG Bordeaux (antenne locale de l’AFUP) organise son premier Coding Dojo dédié à notre langage préféré.

Ouvert à tous, confirmés ou néophytes, le Dojo a pour but d’offrir aux développeurs une opportunité d’apprendre et d’améliorer leur technique de programmation, sans pression, en dehors des projets qu’ils doivent livrer.

Ce Dojo se déroulera le mercredi 4 Juillet à partir de 19h00 dans les locaux d’AlliaForm à la Cité Mondiale :

http://www.alliaform.fr

Le Dojo sera suivi d’un apéro / repas dans le quartier pour ceux qui le souhaitent.

Evenement gratuit ouvert à tous sans inscription.
