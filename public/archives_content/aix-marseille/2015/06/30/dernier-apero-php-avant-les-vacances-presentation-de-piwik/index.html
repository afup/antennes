Dernier Apéro PHP avant les vacances : Présentation de Piwik

Publié le 30/06/2015

Dernière réunion avant la trêve estivale !

Nous vous invitons tous le jeudi 16 juillet pour une magnifique soirée PHP 🙂

Pour cette réunion Maxime Longuet vous présentera PIWIK. C’est un logiciel d’analyse et de mesure d’audience, concurrent direct de Google Analytics. Ecrit en PHP et s’exécutant sur un moteur Mysql, ils vous permet d’avoir votre propre serveur privé de statistique.

Aujourd’hui tous le monde est passé sur Google Analytics et leur confie « gratuitement » ces précieuses données. Piwik est une alternative Libre couvrant la totalité des besoins d’un analyste web et il dispose en plus d’une API Rest pour accéder facilement à toute les statistiques de votre site. On peut alors se servir de ces stats pour apporter des fonctionnalités intéressantes à vos applications. Nous verrons les fonctionnalités de Piwik, son installation, et un exemple d’utilisation de son API en PHP.
