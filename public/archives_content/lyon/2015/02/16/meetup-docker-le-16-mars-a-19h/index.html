Meetup docker le 16 Mars à 19h

Publié le 16/02/2015

Les meetups Docker Lyon et l’AFUP Lyon organisent une conférence sur les pilotes de stockage de Docker, donnée par Jérôme Petazzoni de Docker, Inc.

Vous retrouverez la description et pourrez vous inscrire sur la page du meetup.

Cette présentation aura lieu le lundi 16 mars à 19h dans les locaux de l’INSA, au 6 avenue des Arts, Villeurbanne.
