Retour sur l’apéro PHP du mois de février

Publié le 03/03/2014

Le 29 février dernier une trentaine de développeurs se sont retrouvés à l’Antre-Autre pour assister à une présentation sur Docker lors de l’apéro PHP de février.

Toujours grave du monde au #aperophp de @AFUP_lyon pic.twitter.com/Ou3MLRmt8G

— Olivier Mansour (@omansour) February 25, 2014

Lors de cette présentation, Julien Fusco a abordé différents points :

  • l’intérêt d’utiliser Docker (notamment l’isolation des service pour les scaler plus facilement),
  • les différences avec la virtualisation et le principe des LXC,
  • et enfin, le fonctionnement de Docker (ce qui se passe lors d’un Docker run, le mode de fonctionnement des Dockerfiles).
  • Celle-ci s’en est suivie d’un ensemble de questions. Le sujet a visiblement soulevé les interrogations avec environ 25 minutes de questions/réponses.
